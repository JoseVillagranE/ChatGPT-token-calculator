# ChatGPT-token-calculator
A token calculator for ChatGPT frontend.

## Steps to use

1. Open the ChatGPT frontend in your browser.
2. Click on your profile
3. Go to "Data controls" and click on "Export data".
4. Wait for the export to be ready and download the file (This may take some time and you will receive an email notification when it's ready).
5. Unzip the downloaded content 
6. Edit the corresponding PATH
